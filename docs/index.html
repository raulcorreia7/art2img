art2img

Convert Build Engine ART files to PNG, TGA, and BMP. Supports Duke Nukem 3D, Shadow Warrior, Blood, and other Build Engine games.

GitHub Usage 
# Convert ART files to PNG
art2img convert TILES000.ART --output png/

# Apply transparency and shading
art2img convert TILES001.ART \
  --palette PALETTE.DAT \
  --transparency \
  --shading \
  --output tiles/

Features

ART File Support

Read Build Engine ART files from Duke Nukem 3D, Shadow Warrior, Blood, and other games.

Image Formats

Convert to PNG, TGA, and BMP with RGBA color support and transparency handling.

Palette Processing

Handle PALETTE.DAT files with shade tables, translucency data, and color correction.

CLI & Library

Command-line tool and C++ library for both batch processing and programmatic use.

Cross-Platform

Build and run on Windows, Linux, and macOS.

C++23

Modern C++ implementation with std::expected error handling.

Usage

Build

git clone https://github.com/rcorreia/art2img.git
cd art2img
make setup
make build

Convert

./art2img convert TILES000.ART
./art2img convert TILES000.ART \
  --palette PALETTE.DAT \
  --output output/
./art2img convert *.ART \
  --format png \
  --transparency

API

#include <art2img/api.hpp>

using namespace art2img;

auto art = core::load_art("TILES000.ART");
auto palette = core::load_palette("PALETTE.DAT");
auto image = core::convert_to_image(art, palette);
encode::png(image, "output.png");

About

art2img converts Build Engine ART files to modern image formats. For game modders and developers working with classic Build Engine games.
